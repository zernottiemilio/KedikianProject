Prueba de Codificación Base64

Esta herramienta te permite verificar cómo se codifican las credenciales en base64 antes de enviarlas al backend.

Resultado:

Ejemplo:

Usuario: admin

Contraseña: 123456

Base64:

  • admin → YWRtaW4=
  • 123456 → MTIzNDU2

Request Body:

username=YWRtaW4=&password=MTIzNDU2
